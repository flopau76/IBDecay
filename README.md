# IBDecay
This project is about modeling Identity by Descent (IBD) and how those segments evolve over time between different populations.

## Getting started
The python package can be installed locally using pip:
```bash
git clone https://github.com/flopau76/IBDecay.git
cd IBDecay
pip instal . [simulations]
```

## How to use
The folder `notebook` contains examples, including how to simulate IBD segments using msprime and infer various parameters, such as effective population size, time difference between samples and admixture.
